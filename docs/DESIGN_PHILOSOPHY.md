# Filosofía de diseño

TerMacroMetro transforma información económica dispersa en indicadores transparentes, reproducibles y verificables.

## Principios

- Transparencia metodológica.
- Reproducibilidad.
- Trazabilidad hasta el documento original.
- Neutralidad frente a medios e instituciones.
- Separación entre lenguaje periodístico y realidad económica.
- Automatización acompañada de controles de calidad.
- Evolución incremental sobre versiones estables.
- Apertura para investigación, educación y periodismo de datos.
